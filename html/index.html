Seu site na web.

Crie e publique, de forma rápida e fácil, o seu próprio site na Web usando o iWeb, o Pages e muitos outros aplicativos disponíveis
para Mac OS X.

É extremamente fácil criar o seu próprio site na web com o seu Mac. Quanto terminar de criar o seu site, você perceberá que publicá-lo também é muito fácil.

Abra as Preferências do Sistema e clique em Compartilhamento. Em seguida, selecione Compartilhamento Web.

Pronto, só isso! Seu site agora já está disponível na sua rede privada, em casa ou no trabalho.

Se o seu computador estiver conectado à Internet, seu site também pode ser disponibilizado para seus amigos em qualquer lugar. Basta enviar-lhes o endereço mostrado nas preferências Compartilhamento.

O poder do Apache

O Compartilhamento Web tem como base o servidor web Apache, uma tecnologia padrão do setor que está incluída no Mac OS X. Para obter mais informações sobre o servidor web Apache, consulte o manual do Apache.
